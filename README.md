# FINDSOCK

**Useful CLI tool for developers**

## Features

- 🚀 Fast and efficient
- 📦 Single binary, no dependencies  
- 🔧 Easy to use
- 🔒 Secure by design

## Installation

```bash
git clone https://github.com/simplestar-992/findsock.git
cd findsock
go build -o findsock .
```

## Usage

```bash
./findsock --help
```
