## Findsock

A useful command-line tool for developers.

### Installation

```bash
git clone https://github.com/simplestar-992/findsock.git
cd findsock
go build -o findsock .
```

### Usage

```bash
./findsock --help
```

---

## Quick Start

```bash
./findsock [command]
```
