## ✨ Features

- 🚀 Blazing fast performance
- 💎 Clean and intuitive API
- 🔧 Zero configuration needed
- 🛡️ Secure by default
- 📦 Single binary, no dependencies

## 📥 Installation

```bash
git clone https://github.com/simplestar-992/findsock.git
cd findsock
go build -o findsock .
./findsock --help
```

## 🚀 Quick Start

```bash
./findsock -h
```

## 🧪 Examples

```bash
# Example 1
./findsock status

# Example 2  
./findsock --verbose

# Example 3
./findsock help
```
